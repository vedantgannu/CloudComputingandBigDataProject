Our back end code is divided into the following sections:


1. Lambda functions defined:
	-userFunctionality: responsible for the main restful interactions with our API Gateway set up.
	-LFCreateMedicine: handles the creation functionality of a medicine object and its respective attributes for a particular user.
	-LFUpdateMedicines: handles the update functionality of a medicine for a particular user.
	-LFEnableReminders: handles the enable reminder functionality (on/off)
	-LFGenerateReminderEmail: handles the creation of a reminder email sent out to a patient
	-LFChat: handles the logic for the chat functionality between a patient and a respectiven doctor.
